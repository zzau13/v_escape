# v_htmlescape-bench

Benchmark suite for v_escape crates using Criterion.

## Usage

```bash
cargo bench
```

This will run benchmarks for:

- v_htmlescape
- v_jsonescape
- v_latexescape

## Documentation

- Minimum supported Rust version: 1.85.0 or later
